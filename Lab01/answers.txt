Questions (Page 21):
- What happens if users choose “Cancel”?
-> It will show message dialog "You've chosen: No"
- How to customize the options to users, e.g. only two options: “Yes” and “No”, OR “I do”
and “I don’t” (Suggestion: Use Javadocs or using Eclipse/Netbean IDE help).
-> We can create custom array containing "Yes" and "No" and pass it to showOptionDialog method: 
```java
    Object[] options = {"Yes", "No"};

    int option = JOptionPane.showOptionDialog(null, "Do you want to change to the first class ticket?", "Choose Option", JOptionPane.YES_NO_OPTION, JOptionPane.QUESTION_MESSAGE, null, options, options[0]);
```
